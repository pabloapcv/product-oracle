# Data Sources (v1)

## Amazon (weekly)
Track per ASIN:
- Title, brand, category
- Price, coupon/discount flag
- BSR and BSR category (or rank proxy)
- Review count, rating
- Prime availability / shipping promise
- Seller count (single/multi)
- Listing age (first seen date)
- Images/video count (if obtainable)
Text:
- Review text sample (recent N)
- Bullets/description

## TikTok (daily -> weekly aggregate)
Track per hashtag/keyword:
- Views, videos count
- Growth rates (7d, 14d, 28d)
- Engagement estimates (likes/comments/shares when available)
- Creator adoption count / concentration
Text:
- Comment samples (top/recent)

## Shopify / DTC (weekly)
Track per store domain and product:
- Product title, price, variants
- Availability (in-stock/sold-out)
- Review count (if visible)
- New product events (diff vs prior week)
- Social follower snapshots (optional)
